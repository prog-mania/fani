Программа показывает анимированное изображение человека,
который выглядит, движется, дышит и говорит как живой.

Программа получает в файле files.js:
картинку лица в base64, координаты точек landmarks и их триангуляцию для webGL
и морфы изменения координат точек при деформации лица.
Изменять их можно регуляторами или автоматической анимацией, 
возможно, при произнесении голосом.
Голос получается готовым файлом wav из files.js
или синтезируется на сервере по тексту тоже из files.js.
